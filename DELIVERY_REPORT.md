## 🚀 使用说明

### 快速启动

1. **双击启动脚本**
   ```
   start.bat
   ```

2. **或手动启动**
   ```bash
   # 终端1：启动后端
   python ocr_api.py

   # 终端2：启动前端
   npm run dev
   ```

3. **访问应用**
   ```
   http://localhost:5173
   ```

### 基本使用

1. 上传表格图片
2. 点击"开始识别"
3. 查看识别结果
4. 导出CSV文件
