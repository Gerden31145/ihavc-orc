# LLM集成功能说明

## 功能概述

本系统已成功集成DeepSeek LLM API，为OCR表格识别提供智能增强功能。主要功能包括：

1. **OCR结果智能增强** - 使用LLM纠正识别错误、完善表格结构
2. **表格数据标准化** - 统一数据格式（日期、数字等）
3. **文本表格合成** - 从非结构化文本中提取表格数据
4. **智能纠错** - 自动检测并修复OCR识别错误

## 技术架构

### 后端实现
- **llm_enhancer.py** - LLM增强器核心类
- **ocr_api.py** - 扩展的OCR API，支持LLM增强
- **新增API端点**:
  - `/api/ocr?enhance=true` - 带LLM增强的OCR识别
  - `/api/synthesize` - 从文本合成表格

### 前端实现
- **OcrTable.vue** - 增强的前端界面
- 新增LLM增强开关控件
- 实时显示增强结果和纠错信息

## 使用方法

### 1. 启动服务
```bash
# 安装依赖
pip install -r requirements.txt

# 启动OCR服务
python ocr_api.py
```

### 2. 前端使用
1. 打开前端界面
2. 上传表格图片
3. 启用"LLM智能增强"开关
4. 点击"智能识别"按钮
5. 查看增强后的结果和纠错信息

### 3. API调用

#### 带LLM增强的OCR识别
```bash
curl -X POST "http://localhost:8000/api/ocr?enhance=true" \
  -F "file=@table_image.jpg"
```

#### 文本表格合成
```bash
curl -X POST "http://localhost:8000/api/synthesize" \
  -H "Content-Type: application/json" \
  -d '{"text": "北京大学：理科680分，文科675分..."}'
```

## 性能考虑

1. **API调用延迟** - LLM增强会增加处理时间（约2-5秒）
2. **错误处理** - 增强失败时自动回退到原始OCR结果
3. **缓存机制** - 可考虑添加结果缓存优化性能

2. **增强效果不佳** - 调整提示词或增强策略
3. **服务无法启动** - 检查端口占用和依赖安装
