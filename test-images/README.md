# 测试图片目录

将您的高考分数线表格图片放在这个目录中进行测试。

## 图片要求

- ✅ **格式**: JPG、PNG
- ✅ **清晰度**: 分辨率至少 1024x768
- ✅ **内容**: 清晰的表格结构
- ✅ **光线**: 充足且均匀
- ✅ **角度**: 尽量平整，避免倾斜

## 推荐图片类型

1. **官方发布的高考分数线表格**
2. **学校或教育机构发布的分数线图片**
3. **统计表格截图**
4. **Excel 表格截图**

## 示例图片名称

建议使用描述性文件名：
- `2023_河南_理科一本.jpg`
- `2023_山东_综合类本科.png`
- `2022_全国_各批次分数线.jpg`

## 测试建议

1. 先用简单表格测试（列数少，结构清晰）
2. 逐步尝试复杂表格
3. 记录识别准确率
4. 调整图片质量以提高识别率

## 常见问题

**Q: 为什么识别结果不准确？**
A: 可能原因：
- 图片模糊或分辨率过低
- 表格结构复杂（合并单元格等）
- 字体不清晰或有遮挡
- 光线不均匀

**Q: 如何提高识别准确率？**
A:
- 使用高分辨率图片
- 确保表格边缘清晰
- 避免图片倾斜
- 使用图片编辑工具预处理
